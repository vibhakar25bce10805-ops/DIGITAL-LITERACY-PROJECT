#  Digital Literacy Portfolio

I am Vibhakar Krishna, CSE student from VIT Bhopal interested in programming, problem-solving, and learning new technologies.

## 📚 Project Overview
This repository contains my Digital Literacy Portfolio created as part of my coursework. The project focuses on developing awareness about digital tools, online safety, professional communication, and cyber security.

## 📂 Project Structure

**task-1-presentation/**  
  Digital Literacy Infographic  

 **task-2-portfolio/**  
  GitHub, LinkedIn, and Kaggle profiles  

 **task-3-platforms/**  
  Coding practice and Google Form quiz  

 **task-4-email-etiquette/**  
  Professional emails and social media checklist  

 **task-5-cybercrime/**  
  Cybercrime case study (Phishing) and prevention  

 **report/**  
  Final project report
